## Installation / Local Setup

1. **Clone the repository**
```bash
git clone <repo_url>
cd Zero
```
2. **Install dependencies**
```bash
npm install
```

3. **Start development server**
```bash
npm run dev
```


## Build & Deploy to GitHub Pages

1. **Build the project**
```bash
npm run build
```

2. **Deploy to GitHub Pages**
```bash
npm run deploy
```
